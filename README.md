# YourAnimeList
Client for myanimelist.net
Used technologies:
- Language: Kotlin
- Architecture: MVI
- Network: OkHttp, Retrofit, Coil
- Async: Coroutines, Flow
- Local data: cryptoSharedPreferences, MemoryCache
- Api: https://myanimelist.net/apiconfig/references/api/v2
- DI: Hilt
- Presentation: Compose
